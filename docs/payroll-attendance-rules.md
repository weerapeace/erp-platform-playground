# Payroll Attendance Rules

กฎกลางสำหรับบอกว่าใครต้องเข้าตารางเวลา ใครได้วันหยุดพิเศษ และใครต้องสแกนเวลา

## กติกาหลัก

- งานเหมา / piecework ไม่เข้าตารางเข้างาน และไม่ใช้ import เครื่องสแกน
- รายวัน / รายชั่วโมง เข้าตารางเข้างานได้ แต่วันหยุดพิเศษของงวดไม่จ่ายอัตโนมัติ
- ประจำ / รายเดือน ได้วันหยุดพิเศษตามงวดถ้าวันนั้นตรงตารางทำงาน
- สัญญาที่ติ๊ก `attendance_scan_exempt` ไม่ต้องมี timestamp แต่ยังลงขาด/ลา/หัก manual ได้

## จุดที่ใช้กฎนี้

- ตารางเข้างานรายวันในหน้า Payroll Manual Input
- Import เวลาจากเครื่องสแกน
- API บันทึกสาย/ขาด/ลา/OT
- เครื่องคำนวณเงินเดือน
- Validation ก่อนคำนวณเงินเดือน

## ไฟล์กลาง

ใช้ `lib/payroll-attendance-rules.ts` เป็นแหล่งตัดสินใจหลัก อย่าเขียนเงื่อนไขงานเหมา/รายวัน/ยกเว้นสแกนซ้ำในแต่ละหน้า
